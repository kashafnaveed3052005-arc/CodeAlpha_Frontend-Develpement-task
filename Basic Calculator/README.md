# Task 4 - Calculator 🧮

A simple and interactive calculator built using HTML, CSS, and JavaScript.

## Description

This project is a basic web-based calculator that performs arithmetic operations.
Users can enter numbers using buttons or keyboard and get real-time results.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear display button
- Real-time calculation
- Keyboard support
- Simple user interface design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Concepts Used

### HTML
- Page structure
- Input field
- Buttons

### CSS
- Styling
- Flexbox
- Grid layout

### JavaScript
- DOM manipulation
- Functions
- Event handling
- User input handling

## How to Run

1. Download or clone the repository

2. Open the project folder

3. Open `index.html` in your browser
